## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added support for GitHub Apps authentication for enhanced security
- Introduced theme customization with CSS custom properties
- Implemented automatic API documentation generation from OpenAPI 3.1 specs
- Added support for Mermaid diagrams in documentation pages

## Improvements

- Improved build performance by 40% with incremental compilation
- Enhanced search functionality with fuzzy matching and filters
- Better error messages for configuration validation
- Added keyboard navigation support for accessibility

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Complete rewrite with React-based rendering engine
- Added plugin system for extensibility
- Introduced multi-repository support
- Implemented real-time collaboration features

## Breaking Changes

- Configuration file format changed from JSON to JavaScript
- CLI commands updated (use `sprybear build` instead of `sprybear compile`)
- Theme customization API redesigned

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added dark mode support with automatic theme switching
- Introduced navigation breadcrumbs for better orientation
- Added support for custom domains
- Implemented webhook notifications for documentation updates

## Bug Fixes

- Fixed issue with large repository synchronization
- Resolved memory leak in build process
- Corrected table of contents generation for nested pages

</Update>
